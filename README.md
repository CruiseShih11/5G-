# 5G-移动通信知识
写这篇帖子的动机是：对于很多通信专业的同学来讲，虽然我们总是在讨论着各种各样的通信术语：信道估计，波束成形，波束聚焦，预编码等。但是目前大多数论文与教材中对于整个过程的流程与逻辑并没有深入地提及，因为大多数论文都在研究其中的某一个子问题，进而对这个子问题进行建模如：信道估计中的$y=AH+N$。但是这个建模是如何得到的呢？真实的天线接收到的是什么呢？什么时候进行$IFFT/FFT$的$OFDM$变换呢？这个相应的硬件部分在哪里呢？诚然，不知晓这些具体的问题对于研究单个问题，并不会有很大的影响，但是这对于学生想进一步地深入研究通信系统有巨大的阻碍。笔者根据自己的所学结合各种工具包括AI等，对这一部分内容进行整理，如有错误与补充，请各位$github$上的朋友进行指正。
